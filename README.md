🌦️ Weather App

A Flutter application that provides real-time weather information using external APIs, displaying current conditions and forecasts in a clean and user-friendly interface.


📌 Overview


The Weather App is a dynamic mobile application that allows users to check weather conditions for different locations. It fetches live data from a weather API and displays it بطريقة سهلة وواضحة.
 

This project demonstrates your ability to work with:


🌐 APIs (real-time data fetching)

📊 Data parsing (JSON handling)

📱 Responsive UI in Flutter


Weather apps rely on external APIs to deliver live environmental data such as temperature, humidity, and conditions.


✨ Features

🌍 Search weather by city

🌡️ Display temperature and weather conditions

☁️ Show dynamic weather data from API

⚡ Real-time updates

🎨 Clean and simple UI

🛠️ Tech Stack

Flutter

Dart

REST API (Weather API)

📂 Project Structure
lib/
├── services/      # API & network handling
├── models/        # Weather data models
├── screens/       # UI screens
├── widgets/       # Reusable components
└── main.dart      # Entry point

🚀 Getting Started

Prerequisites

Flutter SDK

VS Code / Android Studio

Internet connection

Installation

git clone https://github.com/AliSayed15/weather_app_setup-main.git

cd weather_app_setup-main

flutter pub get

flutter run

🎯 How It Works

User enters a city name

App sends request to weather API

API returns data in JSON format

Data is parsed into models

UI updates dynamically with weather info
